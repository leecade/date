2. 允许对任意日期或日期段的算法结果进行人工干预(埃及日历是人定的, 随时可能改1天)

比如:

1) 修正 2013-1-1 农历 月份 -1

```
conf.date.fixDate: ["2013-1-1=0|-1|0"]
```

2) 多组修正和日期范围(用 ~ 分隔)

```
conf.date.fixDate: ["2013-1-1~2013-1-11=0|-1|1", "2013-1-12~2013-2-9=0|-1|0"]
```
